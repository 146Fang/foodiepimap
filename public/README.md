# Public Assets

请将以下图片文件放在此目录：

1. **logo-orange.png** - 原图色彩的 Logo（橙红色渐变）
   - 尺寸建议：80x96px 或更高分辨率
   - 格式：PNG（支持透明背景）

2. **logo-green.png** - 绿色版本的 Logo（用于地图标记）
   - 尺寸建议：80x96px 或更高分辨率
   - 格式：PNG（支持透明背景）

## 图片要求

- **Logo（原图色彩）**：使用您上传的橙红色渐变 Logo
- **地图标记（绿色）**：使用绿色版本的 Logo

如果图片不存在，组件会自动使用 SVG 作为后备方案。
